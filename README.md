# 黄金矿工 Gold Miner

经典的挖金矿小游戏 —— 纯前端 HTML5 Canvas 实现，通过 GitHub Actions 自动部署到 GitHub Pages。

## 玩法（iPad / iPhone）

- 点 **右下「放钩」按钮**，或屏幕任意位置都能放钩
- 抓到 **金子 / 钻石** 加分；**岩石** 又重又少分
- 每关在限时内达到 **目标分数** 才能通关
- 越大、越重的物体收钩越慢

| 物品 | 分数 | 收钩 |
|------|------|------|
| 小金块 | 50 | 较快 |
| 中金块 | 100 | 中 |
| 大金块 | 250 | 慢 |
| 小石头 | 10 | 较慢 |
| 大石头 | 20 | 很慢 |
| 钻石 | 600 | 极快 |

## 关卡

共 **20 关**，难度由浅入深：
- L1–3：教学，几乎只有金子，慢速摆钩，200/350/500 分
- L4–10：加入钻石、更多石头，时间收紧
- L11–17：高密度地下，目标分数升至 2700–5200
- L18–20：终关挑战，目标 7000 分

20 关之后无限延伸，目标按 +600/关 继续增加。

## 本地运行

直接用浏览器打开 `index.html` 即可。

## 部署

推送到 `main` 分支即自动通过 GitHub Actions 部署到 GitHub Pages。

需要在仓库 **Settings → Pages → Build and deployment → Source** 中选择 **GitHub Actions**。

部署地址：`https://<username>.github.io/<repo>/`
